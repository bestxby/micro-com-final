Finite State Machine (FSM) Control State Transitions

• INIT (系统上电初始化)：系统硬复位启动，执行外部总线硬件自检（I²C 寻址校验与 ST7796S 控制器握手）。若校验无误，系统程序控制链自动跃迁至自适应学习阶段。

• LEARNING (基准演化学习)：执行前 100 次温度数据帧的连续采样，通过增量递推滤波器求解环境温度基底均值估计量 μbase。此时物理层控制 LED3 指示灯保持高电平常亮，LED2 激活 PWM 硬件通道输出正弦呼吸特征波形。

• MONITOR NORMAL (稳态温度监测)：基准锁存后，系统进入实时监视轮询。在此状态下，若运算出的绝对温差偏差值满足条件 ΔT ≤ 5°C，表明受测物理场处于安全域。此时，LED4 保持常亮以表征稳态，LED2 延续 PWM 低频呼吸指示。

• MONITOR ANOMALY (瞬态异常告警)：当绝对偏差超越安全边界 ΔT > 5°C，或判定传感器发生物理链路丢失（I²C 寻址超时无应答）时，系统立刻关断 PWM 呼吸灯输出，并控制 PC13 GPIO 管脚连同指示灯 LED3 与 LED4 以 5Hz 的周期性频率高频爆闪进行物理警示。通过检测 KEY2 GPIO 外部中断信号可实现人工报警复位。
