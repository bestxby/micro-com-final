系统核心主状态机

System Main Finite State Machine (FSM)

• INIT (初始化)：系统上电复位，校验 I2C 传感器与屏驱动，成功后自动跳转进入学习期。

• LEARNING (基准学习)：采样前 100 次，拟合环境稳定均值 μbase。LED3 常亮，LED2 呈 PWM 缓速呼吸状态。

• MONITOR NORMAL (正常监测)：锁定基准后周期监测。若偏差 ΔT ≤ 5°C，系统处于安全态。LED4 常亮，LED2 PWM 呼吸。

• MONITOR ANOMALY (异常告警)：若温湿度异常突变（ΔT > 5°C）或传感器断线，关闭呼吸灯，PC13 与 LED3/LED4 进入 5Hz 急促爆闪报警模式。按 KEY2 键消音复位。
