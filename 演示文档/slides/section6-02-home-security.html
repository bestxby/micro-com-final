智能居家多级安全防护与状态机

Multi-Level Home Security Logic and Finite State Machine Control

SAFE 状态
触发条件：超声波测距 ≥ 30cm 或处于撤防模式
外设动作：绿灯 (LED4) 平滑呼吸运行，有源蜂鸣器静音。系统处于常规环境监控状态，安全无风险。
WARNING 状态
触发条件：布防模式下，距离介于 15cm 至 30cm
外设动作：黄灯 (LED3) 保持常亮警告，蜂鸣器静音。屏幕数码管开始实时显示侵入距离，提示可能的安全威胁。
ALARM 状态
触发条件：布防模式下，距离 < 15cm
外设动作：红灯 (LED2) 5Hz 高频爆闪，蜂鸣器断续声鸣警报。屏幕弹出警报窗口，并立即触发微信警报推送。

状态机转换规则 (FSM)

• 撤防模式 (Disarmed) → 按 KEY3 / 红外OK → 布防监测 (Armed)
• Armed 状态下，系统持续轮询测距：当超声波距离突变触发特定边界时，直接进行 Safe ↔ Warning ↔ Alarm 的状态机物理映射。撤防操作将强制系统重置为 Safe 状态。
