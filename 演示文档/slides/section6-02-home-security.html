Multi-Level Home Security Logic and Finite State Machine Control

声光防盗多级警戒逻辑

  • SAFE 状态   (超声波测距 ≥ 30cm 或撤防模式)
    外设动作：绿灯 (LED4) 平滑呼吸运行，有源蜂鸣器静音。
  • WARNING 状态   (布防下，距离 15cm 至 30cm)
    外设动作：黄灯 (LED3) 保持常亮，蜂鸣器静音，数码管显示实时距离。
  • ALARM 状态   (布防下，距离 < 15cm)
    外设动作：红灯 (LED2) 5Hz 高频爆闪，蜂鸣器断续声鸣警报，屏幕弹窗警报，并触发微信警报推送 (180s 冷却限制)。

状态机转换规则 (FSM)

• 撤防模式 (Disarmed) → 按 KEY3 / 红外OK → 布防监测 (Armed)
• Armed 状态下，系统持续轮询测距：当超声波距离突变触发特定边界时，直接进行 Safe ↔ Warning ↔ Alarm 物理映射。撤防操作将强制重置为 Safe 状态。
