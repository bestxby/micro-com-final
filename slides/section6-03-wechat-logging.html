ESP8266 WiFi, NTP Clock Sync, ServerChan WeChat Alerts, and SD Logging

物联网通信与存储设计

  • ESP8266 自动断网自愈与 NTP 同步： 通过 AT 指令进行底层握手，断线后自动保存配置重连。上电后拉取中国 NTP 服务器时间。在掉线期间，系统无缝切入本地 10ms 软件 RTC 累加，恢复网络后自动对齐秒级误差。
  • 心知天气 API 轻量解析： 直接使用 strstr 从 TCP 接收环形缓存中匹配特定键值对，舍弃开销巨大的 JSON 库，极大节约了 RAM 空间。
  • 微信警报推送与 180s 冷却保护： 当系统布防遭入侵或温度突变偏差 > 5℃ 时，调用 Server酱 接口向微信推送报警。为防 HTTP 连接拥塞和警报频繁重入，设计了 180 秒的微信发送冷却期。
  • 512B 扇区对齐的 SD 卡日志存盘： 设计了 SD 日志扇区对齐算法。读取或覆写时按照物理 512 字节进行对齐擦写，将含有北京时间戳的事件及环境参数实时持久化归档。
