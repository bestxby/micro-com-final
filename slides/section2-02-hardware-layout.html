AHT20 温湿度 SCL:PA6 / SDA:PB10

BMP280 大气压 SCL:PA6 / SDA:PB10

高精度环境大气压强与内部辅助温度测量，共享 I2C 物理通道。

模拟 I²C 总线 (从机2) 环境大气压测量
BH1750 光强传感器 SCL:PA6 / SDA:PB10

数字式环境照度传感器，测定室内光强（Lx）以作为光照决策本底。

模拟 I²C 总线 (从机3) 室内环境照度测量
HC-SR04 超声波 Trig:PB11 / Echo:PB12

测距防盗模块。通过 GPIO 10us 触发并发射超声波，捕获 Echo 脉宽折算距离。
