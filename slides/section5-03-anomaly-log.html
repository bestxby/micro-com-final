Anomaly Traceback via Circular Ring Buffer Data Logging

定长存储与循环覆盖机制

  • 异常记录数据结构： 当系统检测到环境温度或压强发生剧烈突变时（如超过安全阈值或手动触发），将立即生成一条异常快照记录。采用极简定长结构体打包传感器数据，避免碎片化存储。
    • typedef struct { uint32_t timestamp_s; // 异常发生时的绝对时间戳 float baseline_temp; // 发生异常前的环境基准温度 float current_temp; // 触发警报的瞬时环境温度 float current_press; // 触发警报的瞬时环境压强 } AnomalySnapshot;
  • 无锁环形队列算法： 底层写入算法采用 head = (head + 1) % MAX 的更新机制。当写入指针到达 EEPROM/SD 卡日志区域的尾部时，自动绕回并覆盖最早的数据块，完美规避了频繁擦写单个闪存扇区导致的硬件寿命损耗。
