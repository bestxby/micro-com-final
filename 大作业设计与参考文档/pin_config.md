# STM32环境监测与安全报警系统 - 引脚分配手册 (Pin Configuration Manual)

本手册详细记录了基于 **STM32F103C8T6** 的“环境自适应安全卫士系统”的**最新、无冲突引脚分配方案**。在引入面包板并接入超声波、数码管、蜂鸣器、三色指示灯以及外接防盗按键后，请严格按照本手册进行接线。

---

## 2. 面包板外设连接指南与电气特性

### 2.1 5V 有源分体蜂鸣器 (Active Buzzer)
* **驱动原理**：有源蜂鸣器内部集成振荡源，只需单片机给出高电平即响，低电平即停。
* **物理接线**：
  * `VCC` $\rightarrow$ 开发板 **5V** 供电线（接 5V 声音最响亮，3.3V 可能会微弱）
  * `GND` $\rightarrow$ 面包板公共地（**GND**）
  * `I/O (Control)` $\rightarrow$ STM32 的 **PB15** 引脚（串联限流电阻更为安全）

### 2.2 HC-SR04 超声波传感器
* **工作电压**：**5V**（必须接 5V，接 3.3V 无法工作）。
* **物理接线**：
  * `VCC` $\rightarrow$ 开发板 **5V** 供电线
  * `GND` $\rightarrow$ 面包板公共地（**GND**）
  * `Trig` $\rightarrow$ STM32 的 **PB11** 引脚
  * `Echo` $\rightarrow$ STM32 的 **PB12** 引脚（PB12 容忍 5V 输入，可直连）

### 2.3 面包板按键 (KEY3 - 警戒开关)
* **接线方式**：轻触按键一端接单片机 **PB9** 引脚，另一端直接接公共地 **GND**。
* **工作原理**：STM32 内部配置 PB9 为上拉输入，按键未按下时读到高电平(1)，按下时读到低电平(0)。

### 2.4 TM1637 四位数码管模块
* **物理接线**：
  * `VCC` $\rightarrow$ 开发板 **3.3V** 供电线
  * `GND` $\rightarrow$ 面包板公共地（**GND**）
  * `CLK` $\rightarrow$ STM32 的 **PB13** 引脚
  * `DIO` $\rightarrow$ STM32 的 **PB14** 引脚

### 2.5 红黄绿三色状态指示灯
* **硬件限流**：面包板上的每个 LED 灯（红、黄、绿）必须**串联一个限流电阻**（推荐 220Ω ~ 510Ω）。
* **物理接线**：
  * **红灯**：阳极接限流电阻 $\rightarrow$ STM32 的 **PA0**；阴极 $\rightarrow$ 公共地 **GND**。
  * **黄灯**：阳极接限流电阻 $\rightarrow$ STM32 的 **PA1**；阴极 $\rightarrow$ 公共地 **GND**。
  * **绿灯**：阳极接限流电阻 $\rightarrow$ STM32 的 **PA2**；阴极 $\rightarrow$ 公共地 **GND**。

### 2.6 ESP8266 WiFi 模块 (ESP-01S)
* **工作电压**：**3.3V**。请注意，开发板上的 3.3V 引脚供电能力有限，若因发送射频数据电流不足导致芯片复位，请在模块的 VCC 和 GND 之间并联一个 47uF ~ 100uF 的滤波电容。
* **物理接线**：
  * `VCC` $\rightarrow$ 开发板 **3.3V**（切勿接 5V）
  * `GND` $\rightarrow$ 开发板/面包板公共地 **GND**
  * `CH_PD` (或 `EN`) $\rightarrow$ 接入 **3.3V** 高电平以启用芯片
  * `RST` $\rightarrow$ 接入 **3.3V** 保持正常运行
  * `RXD` $\rightarrow$ STM32 的 **PA9** (USART1_TX)
  * `TXD` $\rightarrow$ STM32 的 **PA10** (USART1_RX)
  * `GPIO0` / `GPIO2` $\rightarrow$ 悬空（正常运行模式）

## 3. 接线检查清单 (双重确认)
在给 STM32 开发板通电运行前，请确认：
1. [ ] **GND 共地**：单片机的 GND 引脚已与面包板的 GND 轨道连接。
2. [ ] **5V 供电线**：超声波模块和有源蜂鸣器的 VCC 已接入开发板的 5V 脚，而不是 3.3V 脚。
3. [ ] **限流电阻**：所有面包板外接 LED 均已接入电阻。
4. [ ] **下载引脚无冲突**：PA13 和 PA14 未接任何外部电路，以防无法烧录程序。
