# STM32-Adaptive-Sentinel: 环境守护卫士与物联网大屏系统

基于 **STM32F103C8T6** 的环境自适应监测、物联网大屏、声光防盗监控与微信报警推送系统。系统采用高内聚、低耦合的模块化设计，适配了 **480×320 TFT LCD 横屏**、**SD 存储卡**、**AHT20/BMP280/BH1750 传感器**、以及面包板外设（**HC-SR04 超声波、TM1637 数码管、有源蜂鸣器、三色指示灯、外接按键**）和 **ESP8266 (ESP-01S) WiFi 模块**。

本项目已全面适配 **Keil v5**，支持 **ARM Compiler 6 (armclang)** 编译环境（已启用 C99 模式）。

## 2. 软硬件开发环境

### 2.1 软件要求
* **集成开发环境 (IDE)**: Keil uVision 5 (编译选项已配置 ARM Compiler 6 / AC6)
* **包含路径 (Include Paths)**: 已配置 `.\User`、`.\RTE`、`.\Hardware`、`.\Library`
* **字符编码 (Encoding)**: 所有源文件及文档均使用 **UTF-8** 编码，防止中文注释在不同系统上乱码。

### 2.2 默认硬件引脚接线 (实验平台: EES-351-MC + 面包板扩展)

为避开总线冲突并精简设计，STM32 排针与底板及面包板外设接线如下：

| :--- | :--- | :--- | :--- |
| **PB6** | 板载 D2 排针 | USART1_TX (接 ESP8266 RXD) | 复用推挽输出 (115200bps) (USART1 重映射) |

## 3. 系统核心功能设计

### 3.1 自动 WiFi 联网
系统上电后通过 USART1 与 ESP8266 (ESP-01S) 进行 AT 指令通信，自动建立 STA 模式并连接到指定的手机 2.4 GHz 热点：
*   **SSID (名称)**：`小小小波`
*   **密码**：`11111111`
模块连接成功后会自动保存配置至 Flash 芯片，保障掉电重启后模块能够由固件快速实现物理层的自动重连。

### 3.2 NTP 网络时间同步与软件 RTC 递增
*   **网络时钟同步**：系统联网后连接 `cn.ntp.org.cn` 同步东八区网络北京时间，自动将英文月份（如 `Jun`）映射为数字格式，转换出标准的 `YYYY-MM-DD HH:MM:SS` 时间。
*   **本地 RTC 递增**：在 1.0s 秒计数块中驱动本地时钟自增。在断网或离线状态下扮演软件 RTC 让时间继续准确跳动；联网后重新与 NTP 校准对齐，避免因高频网络请求引起的系统卡顿。

### 3.3 心知天气（Seniverse）API 户外气象对比
*   系统通过 TCP 单链接接入心知天气服务器（`api.seniverse.com`），通过定位获取当前城市的室外气温及天气状态（如 Sunny、Rainy）。
*   利用极轻量级的 `strstr` 直接在接收环形缓冲区内抓取 `"text":"` 和 `"temperature":"` 字段，省去复杂的 JSON 解析，大幅节约 RAM 开销。

### 3.4 微信警报实时推送（Server酱）与 3分钟 冷却机制
*   **触发场景**：当系统处于**布防状态且检测到距离小于 15cm**（防盗警报），或者 **AI 算法判定当前环境温度突变偏差大于 5℃** 时，单片机会触发 `Check_And_SendWeChatAlert()` 向您的微信发送报警消息。
*   **3分钟 冷却期限制**：为防止警报连续重入导致频繁发送 HTTP 握手，系统设计了 180 秒的微信发送冷却保护。一旦发送成功，后续 3 分钟内警报仍会由本地声光、屏幕提示，但微信不予重复推送，防止消息轰炸和 ESP8266 数据拥堵。

### 3.5 SD 卡网络时间戳日志存储
*   将原来写入 SD 卡的日志开头 `UP:xxxxs` 升级换代为绝对的网络北京时间：
    `2026-06-05 20:15:30 T:25.40C H:50.00% L:120.0lx A:0 M:0`
*   改进了 SD 卡空闲扇区扫描算法。支持同步检查 `'U'` 字符（老日志）与 `'2'` 字符（新北京时间日志），避免新旧日志覆盖和冲突。

### 3.6 第6页 (Page 5: 物联网环境大屏)
*   **大屏排版**：大字号显示实时网络时间戳及 WiFi 热点 SSID 连接状况。
*   **对比看盘**：左侧卡片直观显示室内 AHT20 的实时温湿度；右侧卡片显示同步自外网心知天气的室外温度、天气多态文本及定位城市。

### 3.7 红外遥控无线操纵与 Flappy Bird 映射解码
*   **非阻塞高精度硬件解码**：鉴于主循环中存在大量的传感器读取和较重的绘制任务，本系统采用 **PA9 EXTI 外部中断双边沿触发** 配合 **TIM4 硬件定时器 (1us周期)** 进行 NEC 引导码/数据码的脉宽测量。系统完全无阻塞，且具有极高的解码响应率和稳定性。
*   **多维度指令控制**：
    *   **页面切换**：红外 `Left` (0x07) / `Right` (0x09) 键映射至屏幕翻页，替代板载按键。
    *   **防盗布防**：红外 `OK` (0x15) / `Power` (0x45) 键一键布防/撤防。
    *   **UI主题切换**：红外 `Menu` (0x47) / `Return` (0x43) 键一键切换炫酷暗黑与科技蓝主题。
    *   **像素游戏映射**：在 Flappy Bird 游戏界面（Page 3）中，红外 `Up` (0x40) 或 `2` (0x18) 键直接模拟跳跃按键。用户可使用红外遥控器完全脱机进行遥感操作！
*   **串口诊断助手**：每次解码成功均会将接收到的 8-bit Command 键码以十六进制 `0xXX` 格式打印至 USART1，方便用户对不同型号的红外遥控器键码进行适配。

## 4. 快速运行与验证

1. **设置个人热点**：在手机上开启 2.4 GHz 个人热点，名称设为 `小小小波`，密码为 `11111111`。
2. **下载与编译**：双击 `demo.uvprojx` 打开 Keil 工程，点击 **Rebuild (F7)**。工程可确保 **0 错误、0 警告** 完美编译。
3. **固件烧录**：通过仿真器点击 **Download (F8)** 将 Hex 文件下载至 STM32F103C8T6 板中。
4. **切页与布防**：
   * 使用 KEY_LEFT (PA15) 或 KEY_RIGHT (PA12) 切换页面，或**使用红外遥控器 `Left` / `Right` 键**无线翻页。
   * 按下 KEY1 (PB0) 或**使用红外遥控器 `Menu` / `Return` 键**可以切换 LCD 主题。
   * 按下 KEY2 (PB8) 或**使用红外遥控器 `Up` / `2` 键**在飞鸟游戏页面中让小鸟起飞或跳跃。
   * 按 KEY3 (PB9) 或**使用红外遥控器 `OK` / `Power` 键**进入布防模式。遮挡 HC-SR04 超声波传感器至 15cm 内，观察数码管显示、三色灯同步爆闪、蜂鸣器响起，并检查微信是否收到报警通知。
