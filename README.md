# 🛡️ STM32-Adaptive-Sentinel: 环境自适应守护卫士与物联网大屏系统

> 基于 **STM32F103C8T6** 的高集成度智能家居安全网关。融合了环境多维感知、边缘计算算法、高帧率 TFT-LCD 人机交互、SD 卡持久化归档以及云端物联网微信报警推送机制。

---

## 🌟 项目简介

本项目旨在打破传统嵌入式设备依赖“死板静态阈值”与“高耗能全屏刷新”的设计瓶颈，横跨 **物理感知、边缘计算、人机交互、底层驱动、云端通信** 五大核心层次，构建出一个全天候、全方位、高可用的智能居家安全中枢体系。

系统全面适配 **Keil v5 (ARM Compiler 6 / AC6, C99)**，采用极简高效的纯 C 语言及纯底层寄存器级外设驱动，完美压榨 MCU 性能。

---

## 🚀 核心特性与技术亮点

### 1. 🤖 边缘计算与 AI 算法侧
*   **动态基准自学习**：摒弃静态阈值，创新引入 **一阶 EMA (指数移动平均) 低通滤波去噪**。
*   **温度突变追踪**：内嵌长达 150s 的环境基准自学习追踪模型，当实时室温偏离动态基准线超过 **±5°C** 时，立即触发异常逻辑，极大降低传感器噪声导致的误报率。

### 2. 📱 人机交互与渲染侧 (HUD)
*   **脏矩形 (Dirty Rectangle) 局部刷新**：针对 4.0 寸 480×320 分辨率的 TFT-LCD，自主研发了屏幕增量局部刷新算法，有效规避了 SPI 总线带宽瓶颈，实现了堪比智能手机的 **丝滑高帧率 UI 动画体验**。
*   **双轨视觉主题**：底层支持模块化卡片架构，内置“暗黑”与“科技蓝”双主题硬件级热切换。
*   **彩蛋生态**：在繁重的传感器调度间隙，利用 30ms 局部增量刷新成功移植并运行了流畅的 **Flappy Bird 像素小游戏**。

### 3. 🌐 物联网云端通信侧
*   **网络链路自愈**：上电自动握手 ESP8266 (ESP-01S) 连接 WiFi，并配备底层轮询重试调度，在弱网或断网重连环境下实现 **零挂死率**。
*   **NTP 时钟与天气拉取**：开机自动同步高精度北京时间（配合本地 1s 心跳维持软件 RTC），并通过 TCP 极轻量级解析心知天气 API，获取室外温度与天气实况。
*   **微信防抖警报**：一旦超声波侦测到异常靠近 (<15cm) 或温度剧烈突变，立刻通过 Server酱 向微信推送图文警报。系统内置 **180s 物理冷却防抖软锁**，坚决拦截重复拥塞的 HTTP 报警请求。

### 4. 💾 底层驱动与存储优化
*   **非阻塞红外解码**：采用 PA9 EXTI 外部中断双边沿触发 配合 TIM4 (1us精度) 进行 NEC 红外遥控非阻塞微秒级解码，彻底释放 CPU。
*   **SD 卡对齐归档**：针对 FAT32 特性设计了 **512B 扇区对齐的无损日志写入算法**。结合无锁环形队列，将带有北京时间戳的异常快照永久归档。

```text
│   ├── bmp280.c/h      # BMP280 气压/温度传感器驱动
│   ├── bh1750.c/h      # BH1750 光强度传感器驱动

├── Library/            # 库层：系统启动及底层寄存器映射文件
├── RTE/                # Keil RTE 运行时环境
├── Scripts/            # 辅助工具脚本 (如 SD 卡日志读取 Python 脚本)
└── demo.uvprojx        # Keil uVision5 工程文件
```

## 🔌 硬件引脚映射 (Pinout)

为了避开总线冲突并精简布线，STM32 排针与 EES-351-MC 底板及外设接线严格定义如下：

| STM32 引脚 | 物理连接对象 | 核心功能角色 | 底层驱动模式 / 备注 |
| :--- | :--- | :--- | :--- |
| **PB6 / PB7** | ESP8266 (RXD/TXD) | **物联网通信中枢** | USART1 重映射 (115200bps / 环形接收中断) |
| **PA0 / PA1** | LED2(红) / LED3(黄) | **警报灯 / 学习灯** | TIM2_CH1 / CH2 复用推挽输出 |
| **PA2** | LED4(绿) | **守护运行指示灯** | TIM2_CH3 复用推挽输出 (带呼吸效果) |
| **PB0** | BTN1 按键 (S0) | **屏幕主题切换** / 自愈重置 | GPIO 上拉输入 (KEY1) |
| **PB8** | BTN2 按键 (S1) | **飞鸟跳跃** / 游戏开始 | GPIO 上拉输入 (KEY2) |
| **PA15 / PA12** | BTN3 (S2) / BTN4 (S3) | **页面左/右切换** | GPIO 上拉输入 (KEY_LEFT / KEY_RIGHT) |
| **PB9** | 面包板外接独立按键 | **一键布防/撤防** 警戒开关 | GPIO 上拉输入 (KEY3) |
| **PB15** | 分体有源蜂鸣器 | **防盗/高温 警报长鸣** | GPIO 推挽输出 (高电平触发) |
| **PB13 / PB14** | TM1637 数码管 (CLK/DIO)| **本地实时气温显示** | 推挽输出 / 开漏双向输出 |
| **PB11 / PB12** | HC-SR04 超声波 (Trig/Echo)| **防盗测距雷达** | 推挽输出 (10us) / 下拉输入脉宽测定 |
| **PA6 / PB10** | 共用 I²C (SCL/SDA) | **AHT20 / BMP280 / BH1750**| GPIO 模拟开漏输出总线 |
| **PB4 / PB3 / PB5**| SD 卡 (CS / MOSI / SCK)| **日志文件归档 (SPI)** | GPIO 推挽 (需解除 JTAG 占用) |
| **PA11** | SD 卡 (MISO) | **日志读取返回** | GPIO 上拉输入 |
| **PA9** | HX1838 红外接收器 (DAT) | **全功能红外无线遥控** | GPIO 上拉输入 (EXTI9 + TIM4 微秒解码) |
| **PA3/PA4/PA5/PA7/PA8/PB1** | TFT-LCD 接口 | **交互大屏驱动** | 模拟 SPI 组 |

## 🛠️ 快速上手与编译部署

1. **配置测试热点**：
   在手机或路由器上开启 2.4 GHz WiFi 个人热点：
   * **SSID (名称)**：`小小小波`
   * **密码**：`11111111`
   *(网络用于获取 NTP、天气及微信推送，若无法联网，系统会自动降级为离线保护模式)*

2. **编译工程**：
   双击打开根目录下的 `demo.uvprojx`，确保已安装 ARM Compiler 6。点击 **Rebuild (F7)**。工程配置已严格优化，确保 **0 错误、0 警告 (0 Error, 0 Warning)**。

3. **固件烧录**：
   连接 ST-Link 或 DAP-Link 仿真器，点击 **Download (F8)** 将 Hex 固件烧录至核心板。

## 🎮 终端操作与功能演示指南

设备上电后，会自动进行传感器初始化并尝试连接 WiFi。主界面（Page 0）为综合仪表盘。

### 1. 物理按键与红外遥控双驱操作
您可以直接操作开发板按键，或者使用配套的 **NEC 协议红外遥控器** 实现完全“脱机无线操纵”：
*   **翻页切换**：按 `BTN3 / BTN4` (或遥控器 `◀ 左` / `右 ▶` 键)，在 6 个核心交互页面间无缝滑动。
*   **主题切换**：按 `BTN1` (或遥控器 `MENU` / `Return` 键)，瞬间切换暗黑 Neon / 科技蓝 视觉主题。
*   **一键布防**：按外接 `KEY3` (或遥控器 `OK` / `Power` 键)，进入安防警戒模式。屏幕顶部会出现 `SYSTEM ARMED` 状态栏。

### 2. 安防警报与突变测试
*   **超声波防盗触发**：在布防模式下，用手遮挡超声波传感器至 **15cm** 以内。系统会立即触发三色灯爆闪、蜂鸣器长鸣，并且您的绑定微信会瞬间收到“防盗警报”图文推送。
*   **环境温控测试**：对 AHT20 温湿度传感器进行加热（如手指长按捂热）。当温度偏离系统过去 150 秒学习到的基准温度超过 5°C 时，触发高温异常保护。
*   **异常归档检查**：拔下 SD 卡，插入电脑，即可在根目录下看到包含精确北京时间戳的异常数据快照日志。

### 3. 娱乐彩蛋
*   翻页至 **Page 3 (Flappy Bird)**。
*   按下 `BTN2` (或遥控器 `▲ 上` / `数字 2` 键) 控制小鸟起跳。在避开水管的同时，体验脏矩形算法带来的 60FPS 无拖影极限渲染！
